## 📋 CMS Features & Sections

### 1. **Overview Dashboard** 📊
- Quick stats showing gallery items, projects, courses, and products
- Transaction ledger overview
- Storage usage summary

### 2. **Hero Section** 🦸
Customize your home page hero banner:
- **Badge** - Professional designation badge
- **Title Lead** - Main title keyword
- **Rotating Words** - Dynamic text that cycles (e.g., "Multi-Disciplinary Technologist")
- **Name** - Your full name
- **Description** - Professional tagline
- **Call-to-Action Buttons** - Primary and secondary CTAs
- **Hero Image** - Upload your profile image
- **Stats** - Display key metrics (Students, Projects, Success Rate, etc.)

### 3. **Gallery Management** 🖼️
Showcase your creative work:
- Add/edit gallery items with images or videos
- Categorize by type (3D Design, Photography, CAD, etc.)
- Add detailed descriptions and specifications
- Support for both image and video media

### 4. **Projects Showcase** 🚀
Display your portfolio projects:
- **Project Title** - Name of the project
- **Category** - web, software, security, or 3D
- **Technology Stack** - List of technologies used
- **Description** - Project overview
- **Case Study** - Detailed story about the project
- **Demo URL** - Link to live demo or repository
- **Featured Image** - Project thumbnail

### 5. **Academy/Courses** 📚
Manage your educational content:
- Course titles and descriptions
- Course syllabus and structure
- Badge and pricing information
- Lecture listings
- Free vs paid courses

### 6. **Store Products** 🛍️
Manage digital products in your store:
- Product names and descriptions
- Pricing and categories
- Product details and preview type
- Support for software, 3D assets, themes, and graphics

### 7. **Custom Sections** ✨
Add custom content to any page:
- Create sections for specific pages
- Add text, images, and videos
- Toggle visibility on/off
- Support for eyebrows and body copy

### 8. **🎨 Branding Editor** (NEW!)
Professional logo and header management:

**Logo Upload:**
- Upload your professional logo (PNG or JPG recommended)
- Choose logo size: Small, Medium, or Large
- Logo displays in the header next to your brand name

**Header Customization:**
- **Background Color** - Solid color for header background
- **Header Gradient** - CSS gradient for dynamic header styling
- **Brand Name** - Your company/personal brand name
- **Show Brand Name Toggle** - Display or hide the brand name
- **Brand Subtitle** - Company tagline (e.g., "Software House")
- **Show Subtitle Toggle** - Display or hide the subtitle

**Live Preview:** See your header styling in real-time as you edit

### 9. **🌈 Theme Customizer** (NEW!)
Full color customization (like WordPress theme builder):

**Primary Colors:**
- **Primary Color** - Used for main buttons, CTAs, and highlights
- **Secondary Color** - For backgrounds and secondary elements

**Accent Colors:**
- **Accent Color** - Badges, highlights, special elements
- **Accent Background** - Background for accent elements

**Text Colors:**
- **Dark Text** - Main content text
- **Light Text** - Text on dark backgrounds

**Background Colors:**
- **Main Background** - Primary page background

**Features:**
- Copy hex codes to clipboard
- Live color preview
- Reset to default theme option
- Visual color palette summary

### 10. **⚙️ Element Customizer** (NEW!)
Customize UI elements and behavior (like Elementor):

**Layout & Spacing:**
- **Page Spacing** - Compact, Normal, or Spacious
- **Header Behavior** - Top or Sticky positioning

**Visual Styles:**
- **Corner Style** - Sharp, Rounded, or Pill-shaped borders
- **Button Style** - Solid, Outline, or Ghost buttons
- **Card Shadow** - None, Light, Medium, or Heavy shadows
- **Base Font Size** - Small, Normal, or Large text

**Features & Behavior:**
- ✅ Enable/Disable Animations
- ✅ Show/Hide Breadcrumbs navigation
- ✅ Show/Hide Scroll to Top button

**Live Preview:** See all styling changes instantly

## 💾 Backup & Export/Import

### Export Backup
1. Click the **Export** button at the top
2. Your website content and products are downloaded as JSON
3. Keep backups regularly for safety

### Import Backup
1. Click the **Import** button
2. Select a previously exported JSON file
3. All data is restored
4. Click **Save Changes** to apply

## 🚀 How to Use the CMS

### Step-by-Step Guide:

#### 1. **Access the Admin Panel**
- Click **Admin** in the main navigation
- Log in with credentials provided above

#### 2. **Edit Your Branding**
- Go to the **🎨 Branding** tab
- Upload your professional logo (recommended: 300x300px PNG)
- Customize header gradient and colors
- Enable/disable brand name and subtitle display
- See live preview

#### 3. **Customize Colors (Theme)**
- Go to the **🌈 Theme** tab
- Click on any color to open the color picker
- Enter hex codes or use the picker
- Copy colors to clipboard
- Use "Reset to Default" if needed

#### 4. **Customize Elements**
- Go to the **⚙️ Elements** tab
- Choose spacing style (compact/normal/spacious)
- Set corner radius for all elements
- Pick button and card shadow styles
- Enable/disable animations and features

#### 5. **Edit Content**
- Click on specific tabs (Hero, Gallery, Projects, etc.)
- Edit text, descriptions, and details
- Upload images and videos
- Reorder items using arrow buttons
- Add new items with the + button

#### 6. **Save Your Changes**
- Click the **Save Changes** button (top right)
- Your changes are immediately applied
- Browser storage is used (local to this device)

## ⚡ Pro Tips

1. **Regular Backups** - Export your CMS data weekly to prevent data loss
2. **Color Testing** - Test your theme colors on different pages for consistency
3. **Logo Sizing** - Use medium size for balanced header appearance
4. **Animations** - Disable if you have slower devices or prefer minimal design
5. **Spacing** - Use "spacious" on content-heavy pages for better readability

## 🛠️ Technical Details

### Storage
- All data is stored in **browser localStorage**
- Maximum recommended content size: 5-10MB
- Export regularly for backup

### Browser Support
- Works on all modern browsers (Chrome, Firefox, Safari, Edge)
- Requires JavaScript enabled
- Uses IndexedDB for enhanced storage

### Customization Limits
- Logo: Max 5MB, recommended PNG format
- Colors: Any valid CSS hex color
- Elements: Customizations apply globally
