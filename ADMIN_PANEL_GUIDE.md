## 💾 Saving Your Work

### **Automatic Save**
- All changes save automatically to browser storage
- No risk of losing work

### **Manual Export**
1. Click "Export Config" button
2. Saves as `website-config.json`
3. Keep as backup

### **Import Configuration**
1. Click "Import Config" button
2. Select previously exported `.json` file
3. All settings restored

### **Reset to Default**
- Click "Reset to Default" to start fresh
- This only applies if something goes wrong

## 🎨 Color Customization

### **How Color Pickers Work**
- Click the colored square to open color picker
- Or paste hex color codes (e.g., #FACC15)
- Supported formats:
  - Hex: #FACC15
  - RGB: rgb(250, 204, 21)
  - Color names: yellow, blue, etc.

### **Recommended Colors**
- **Primary (Brand)**: #FACC15 (Yellow)
- **Background**: #08080C (Dark)
- **Text**: #FFFFFF (White)
- **Accent**: #ec4899 (Pink)

## 📸 Logo Upload Tips

**When uploading logo via admin panel:**

1. **Size**: Keep logo image 200x200px or smaller
2. **Format**: Use PNG with transparent background (recommended)
3. **Placement**: 
   - Via logo editor: Text + Icon approach works best
   - Via image: Use the Background Color setting
4. **Alignment**: Text and Icon are centered by default

**If logo looks weird:**
- Use the emoji/icon approach instead
- Add a background color for better framing
- Adjust font size to match content

## 🚫 Troubleshooting

### **Changes not saving?**
- Check browser console for errors
- Try refreshing the page
- Clear browser cache if needed

### **Logo looks cut off?**
- Use the Font Size slider to adjust
- Make logo background color visible
- Test in both desktop and mobile views

### **Website still showing old colors?**
- Refresh browser (Ctrl+R or Cmd+R)
- Clear browser cache
- Try incognito/private browsing

### **Can't open admin panel?**
- Look for yellow "Builder" button in bottom-right
- Should be visible on all pages
- Try scrolling down if not visible

## 🎓 Best Practices

### **Logo Design**
- Keep it simple and recognizable
- Use high contrast colors
- Emoji logos work great for tech companies

### **Colors**
- Use 2-3 colors maximum
- Ensure good contrast for readability
- Test on mobile before finalizing

### **Sections**
- Use clear, descriptive headings
- Keep descriptions concise
- Organize logically from top to bottom

### **Mobile**
- Test every change on mobile
- Keep text readable (14px minimum)
- Large touch targets for buttons
